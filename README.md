# 🚀 Overview

**ClawCartel** is a **multi-agent product development platform** that turns a simple idea into a launchable Solana project.

Built on an **OpenClaw-powered agentic architecture**, ClawCartel uses **skill-based autonomous agents** that collaborate to plan, design, and build projects automatically.

These agents use specialized **skills** to perform tasks such as idea analysis, architecture planning, PRD generation, and code creation. The result is a fully orchestrated workflow that moves a project from **idea → planning → approval → build → launch**.

The entire process is visualized in a **single execution timeline**, allowing users to track discussions, generated artifacts, and project progress in real time.

---

# ✨ Features

### 🧠 Multi-Agent Collaboration
Multiple AI agents collaborate to analyze ideas, plan architecture, and coordinate development tasks.

### 📄 AI-Generated PRD
Agents automatically generate a **Product Requirements Document (PRD)** based on their discussion and planning process.

### ✅ Human Review & Approval
Users review and approve the generated PRD before the development phase begins.

### 📦 Automated Project Generation
Generate complete project artifacts automatically:
- Frontend
- Backend
- Solana smart contracts (when required)

### 📱 Mobile-First Experience
ClawCartel is designed as a **mobile application**, allowing users to launch projects directly from their phone.

### 📊 Execution Timeline
Track every step of the agent workflow in a **single run timeline**, including:
- agent discussions
- PRD generation
- artifact creation
- execution progress
- deployment status

# 🔁 How It Works

1. **Create a project**

   Enter a simple idea describing the product you want to build.

2. **Agent discussion & planning**

   AI agents analyze the idea and collaborate to design the system architecture.

3. **PRD generation**

   Based on the discussion, the agents generate a **Product Requirements Document (PRD)** describing:
   - project scope
   - features
   - system architecture
   - implementation plan

4. **User review & approval**

   The PRD is presented to the user for **review and approval** before development begins.

5. **Artifact generation**

   After approval, agents generate the required project components:
   - frontend
   - backend
   - smart contracts

6. **Project execution**

   The platform runs the generated workflow and prepares the project for launch.

7. **Track the timeline**

   Users can monitor the entire process through a unified execution timeline.

# 🧠 Architecture

ClawCartel is built around a **multi-agent orchestration system** where agents collaborate to design and execute product development workflows.

Core agent roles include:

- **Planner Agent**  
  Breaks down the user idea into structured product goals.

- **Architecture Agent**  
  Designs system structure and technical approach.

- **PRD Agent**  
  Generates a structured Product Requirements Document.

- **Developer Agents**  
  Generate code for frontend, backend, and smart contracts.

- **Execution Engine**  
  Runs the workflow and records the project timeline.

---

# 🧠 AI & Agent Framework

ClawCartel uses an **agentic architecture powered by OpenClaw**, where autonomous agents collaborate using structured **skills** to execute complex product development workflows.

Each agent is equipped with specialized **skills**, enabling them to perform tasks such as:

- idea analysis
- architecture planning
- PRD generation
- code generation
- workflow execution

These agents communicate through structured discussions and coordinate actions to move a project from **idea → planning → approval → build → launch**.

This skill-based agent system allows ClawCartel to automate tasks that normally require an entire development team.
